# Chrome Web Store assets

Generated with:

```powershell
npm run store:assets
```

Upload mapping:

- `icons/icon-128.png` — Store icon, 128x128 PNG with 16px transparent padding.
- `screenshots/*.png` — Five full-bleed product screenshots, 1280x800 PNG.
- `promo-small-440x280.png` — Required small promotional tile.
- `promo-marquee-1400x560.png` — Optional marquee promotional image.
- `preview-contact-sheet.png` — Local QA preview; do not upload to the store.

The screenshot data is synthetic and contains no real user information.
